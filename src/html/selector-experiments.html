Adjacent Sibling Selector, A + B
  1. Style i1 if it is the first child (i:first-child)
  2. Style div1 if there is an <i> element before div1 using the Adjacent Sibling Selector (A + B)
  3. Style the last <i> element if one is present (i:last-child)
  4. Style paragraph and div3 if an <i> element is the last element
  5. There is no such ting as a preceding sibling element selector, e.g. A - B, so this is not possible with CSS
